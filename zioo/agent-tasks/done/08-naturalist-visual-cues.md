# Epic: Naturalist Visual Cues & Vibe Enhancement

The goal is to elevate the "Clinical Naturalism" aesthetic with "bold and delightful" naturalist cues. The interface should feel more "vibey" and premium.

## Feature: Organic Backgrounds & Textures

- [x] **Research & Generate Assets**: Use `generate_image` to create high-quality, slightly opaque, dark plant-themed background assets. Think dark moss, deep green fern fronds with delicate dew, and soft botanical shadows.
- [x] **Implementation**:
  - Add these backgrounds to the `Store` page and `Hero` section.
  - Ensure they are "slightly opaque" (e.g., `opacity-20` on a dark backdrop or blending with the background color).
  - Avoid using tailwind color shorthands like `bg-primary/80`. Use full HSL/Hex or CSS variables.
- [x] **Vibe Check**: The transitions between sections should feel fluid and organic.
