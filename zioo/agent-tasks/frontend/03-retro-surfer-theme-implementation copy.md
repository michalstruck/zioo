# Epic: "Chill Surfer" Retro Design Implementation

## Background Context

We are introducing a new design aesthetic while explicitly keeping the old design intact via the design switcher.
The new aesthetic is a **chill surfer vibe, slightly retro and hippie**. It should clearly resonate with the new wave of GenZ surfers who long for gone times of summer and freedom. Think sun-faded colors, organic wavy shapes, vintage typography, and a relaxed, easygoing mood.
